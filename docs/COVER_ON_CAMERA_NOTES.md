# 出镜封面笔记（测试结论，尚未做成 skill）

## 竖屏（已验证方向）

- 口播成片人脸多在画面**中心**。
- 标题大字放在 **脸下方（优先）** 或 **头顶上方**，避免压在五官上。
- **字号要大**，缩略图第一眼可读主题。
- 做法：抽帧 → 画面为底 → 渐变托底 + 大字 → 导出 PNG。
- 字体：见 `FONT_LICENSE.md`。

## 横屏（待有成片再做）

- 暂无横屏口播成片时不强制产出。
- 有成片后：
  - **整张脸完整入画**（不要默认左右对半：左字右脸）。
  - 大字叠加，**不挡脸**（脸旁/上/下安全区均可）。

## 与不出镜

- 不出镜方案另议；视频默认优先出镜抽帧流。
