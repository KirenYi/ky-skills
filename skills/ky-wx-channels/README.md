# ky-wx-channels

Agent workflow skill for WeChat Channels (视频号) and Official Account RSS.

Does **not** ship a downloader binary. Orchestrates the open-source project:

- https://github.com/ltaoo/wx_channels_download

## Typical flow

1. Install upstream release (admin on Windows).
2. Open WeChat PC Channels page, use injected download button.
3. Optional: Official Account RSS while article page stays open.

See `SKILL.md` for full Agent instructions.
