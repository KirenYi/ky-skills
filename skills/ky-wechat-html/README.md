# ky-wechat-html

将 Markdown 转成可粘贴到微信公众号后台的单文件 HTML。

- 调用：`/ky-wechat-html`
- 只做排版，不改观点
- 内置多套排版风格

## 在 Agent 里

```text
/ky-wechat-html
把这篇文章排成公众号 HTML
```

或指定：

```text
/ky-wechat-html essay 风格，文件 path/to/article.md
```

## 安装

在 `ky-skills` 仓库根目录：

```bash
./scripts/install-links.sh ky-wechat-html
```
